# MobileFilter Dump
This repo is for analyzing Lightspeed's internal software for the mobile app. All the tools you need are in here, and go to ANALYSIS.md for information on the actual file. You can find the purpose of the tools in the .md file, as well as descriptions of the actual files themselves. This was made and decrypted entirely by Claude. The only change from the actual Relay Smart Agent's app is the lack of Fairplay in the Mach-O Binary file. You can find the actual unchanged ipa file at https://github.com/groffstock/Relay-Smart-Agent-ipa.
